conneg.info

A tiny website and domain about Internet Content Negotiation, in particular, the extension known as Content Netgotiation by Profile (ConnegP).

The main resources within this domain are:

  • W3ID Profiles Catalogue - a catalogue of formal _Profiles_ characterised according to the Profiles Catalogue
  • Mediatypes Service - currently under repair! - a Linked Data API giving LD access to IANA's Media Types info
